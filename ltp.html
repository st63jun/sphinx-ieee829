2. レベルテスト計画¶

レベルテスト計画 (Level Test Plan) は、個々のテストの対象範囲、手法、（人的、物的、ソフトウェア的な）リソース、テスト実施のスケジュールについて記述する文書です。

テストすべき機能や、テスト実施のために完了すべきタスク、個々のテストの責任者、テスト実施によって発生するリスクをここで定義します。

2.1. 記法¶

.. ieee829:ltp:: id¶

レベルテスト計画のセクションを作成します。セクションには id で文書識別番号を設定します。たとえば id に 1 を指定すると、このセクションに LTP-1 というユニークなIDが付与されます。

:title
文書のタイトルを指定します。指定しない場合はIDがタイトルになります。
:ieee829:ltp:¶

レベルテスト計画を参照します。:ieee829:ltp:`1` で LTP-1 を参照します。
